переменная LANG
отвечает за язык системы (локализацию)
LANG=en_US.utf8

VAR сохраняет другие переменные для передачи в другие программы
VAR=$(whoamI)
VAR=whoamI`
VAR=$USER

переменная PATH хранит список папок в которых надо искать программу
папка в которой находится программа будет находиться в списке PATH
программа ищет программу с именем которой вы написали будет искать её в PATH
запустит первую найденную программу с таким именем
если изменим PATH то могут перестать запускаться все программы

в рамках линукса файл является исполняемым если у него
в частности исполняемыми являются скрипты
после двух символов пишется интерпретатор который умеет работать с такими файлами
#!/bin/python3
#!/bin/bash

как запустить скрипт?
bash b.sh
можем сначала сделать файл исполняемый
как это сделать
chmod +x b.sh

как сказать что я хочу запустить файл находящейся в текущей папке
поставить точку!
./b.sh

для измения параметров пишем
$1 - первый параметр
$0 - ./b.sh
$@ - все параметры больше 0
$# - количесвто параметров

есть ещё параметр $! - можно узнать какое число вернула программа
это из C и С++
если 0 - программа отработала корректно
если не 0, значит ошибка

Когда запускаем ворд
то запускается программа с первым параметров идет имя файла которого мы запустили
